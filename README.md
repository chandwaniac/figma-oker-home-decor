# Oker Home Decor Mobile App

A pixel-perfect React implementation of the Oker Home Decor mobile shopping app, built from Figma designs.

## Features

- **Home Page**: Inspiration feed and Shop tabs with product grid
- **Product Details**: Image carousel, color/size selectors, quantity counter
- **Search**: Recent searches, popular categories, brand discovery
- **Favourites**: Products and collections management
- **Cart**: Full shopping cart with checkout flow
- **Bottom Navigation**: Matching Figma design exactly

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS v4
- Vite
- React Router DOM v7
- Lucide React Icons
- Framer Motion

## Design Tokens

- Primary Color: #F5A623 (Amber)
- Font: Inter
- Mobile-first responsive design (max-width: 430px)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
